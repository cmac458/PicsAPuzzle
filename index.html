Welcome to PicsaPuzzle Git

This is a simple app created using mainly with Dart as a learning aid in order to learn basic concepts of Dart language and syntax. The application also utilizes Twitter Bootstrap library for some of the UI components. Try it out here The beauty of this style of learning app is that it can continue to grow as my knowledge of Dart increases.

Idea

The idea was to create an application that could continue to grow with my knowledge of Dart, It had to be something fun to keep me interested and that had the potential to get more complex.
